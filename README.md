## ⌨️ Keybinds
| Keys | Action |
|---|---|
| `Win + Q` | Kill focused window |
| `Win + D` | Open app launcher |
| `Alt + Z` | Spawn terminal |
| `Alt + ←` / `Alt + →` | Switch between windows |
| `Alt + Tab` | Switch between windows |
| `Alt + ↑` / `Alt + ↓` | Switch between workspaces |
| `Win + [1-9]` | Jump to workspace N |
| `Win + Shift + [1-9]` | Move window to workspace N |
| `Win + Shift + [` / `]` | Move window to another monitor |
